# Телеграм-бот

```
Идея данного программного продукта заключается в общении с ботом, который позволяет хранить 
фотографии пользователя с описанием, его важные ссылки и список любимых песен. 
```

```
/start: приветствие бота
- /songs: список песен
- /links: список ссылок
- /pics: фотографии
- - /show_pics: показать фотки 
- - /add_pic: добавление фотки
- - - /cancel: отмена 
- - /delete_pic: удаление фоток
- - /back: назад
```

```
Заостряя внимание на фотографиях, стоит подметить, что помимо просмотра всех 
фотографий, также возможно добавление и удаление изображений.
```
